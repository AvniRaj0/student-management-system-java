# Student Management System (Java)

A simple console-based Java application to manage student records.
This project is designed for beginners to understand and apply core Java concepts.

## Features
- Add a student
- Display all students
- Search student by roll number

## Concepts Used
- Object-Oriented Programming (classes and objects)
- ArrayList for dynamic data storage
- User input handling using Scanner
- Basic control structures (loops and conditionals)

## How to Run
1. Clone or download the repository
2. Open the project in any Java-supported IDE (IntelliJ / Eclipse / VS Code)
3. Compile and run the main Java file
4. Follow the console menu to manage student records

## Use Case
This project can be used as:
- A beginner Java mini-project
- A reference for academic assignments
- Practice material for understanding Java fundamentals

## Tech Stack
- Java
- ArrayList
- Scanner

## Note
This project was built as a learning-focused mini project and can be extended with features like file handling or database support.
